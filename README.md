## Top 8 Projects

### 🏦 1 · Eurobank Financial Analysis 2022–2024
**Python · SQL · SQLite · Power BI · DAX**

Full pipeline built from scratch — no pre-cleaned datasets. Pulled raw data straight from Eurobank's official Annual Report PDFs, ran it through ETL, loaded it into SQLite, and built a Power BI dashboard with custom DAX measures.

Key results: NII up 72% over 2 years, Cost-to-Income down from 46.8% → 33.0%, ROE at 16.9% — above the European banking average.

→ [View Repo](https://github.com/papastergiousp-maker/ETL_Pipelines/tree/main/eurobank-financial-analysis)

### 🛢️ 6 · Retail Finance Sales Analysis
**Python · pandas · NumPy · Matplotlib · Plotly · Seaborn**

End-to-end ETL on a real liquor sales dataset (2016–2019) — date filtering, cleaning, market share and geographic distribution analysis. Built multiple visualizations including store-level % share and top zip code breakdowns.

Top 15 stores captured the majority of total sales volume — geographic zip code analysis revealed key high-demand distribution zones.

→ [View Repo](https://github.com/papastergiousp-maker/Python_Financial_Data_Analytics/blob/main/Retail_Analysis/FINANCE_LIQUOR_SALES.ipynb)

### 🧪 7 · Production Quality & Financial Impact Analysis
**Python · pandas · NumPy · Matplotlib · Seaborn**

Simulated 480 production batches using Normal Distribution, applied business logic (pH thresholds) to classify each batch and calculate the direct financial cost. Built as a financial reporting tool — not just a visualization.

Total financial impact: €76K across 480 batches — €56K in corrections, €20K in critical waste.

→ [View Repo](https://github.com/papastergiousp-maker/Industrial-Quality-Control-Financial-Risk-Modeling/blob/main/INDUSTRIAL%20ANOMALY%20DETECTION.ipynb)
